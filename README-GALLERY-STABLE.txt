Gold Hunter V19 Stable Gallery

Fixes:
- Mobile Show More grid is smaller (Apple Photos style).
- Gallery count is based on manifest.json only, so it will not show fake 90/192 items.
- Removed post-manifest scan that caused photos to disappear after cache refresh.

How to add Results:
1. Put file in assets/results/
2. Add the filename to assets/results/manifest.json
3. Put newest item at the TOP if you want it to appear first on homepage.

How to add Reviews:
1. Put file in assets/reviews/
2. Add the filename to assets/reviews/manifest.json
3. Put newest item at the TOP if you want it to appear first on homepage.
