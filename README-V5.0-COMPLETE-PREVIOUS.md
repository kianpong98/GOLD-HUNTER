# Gold Hunter V5.0 — Complete Previous Coverage

- BLS remains the primary source for CPI, PPI and labour data.
- FRED provides an independent fallback for every numeric event type.
- Partial BLS/FRED refreshes merge per metric and never erase a previously successful value.
- Runtime caches are retained for 30 days and refreshed automatically.
- Previous remains automatic; Admin only edits Forecast.
- Old official observations can populate Previous but can never populate a new release Actual.
