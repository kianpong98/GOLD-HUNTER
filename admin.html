Gold Hunter Gallery Manager

This page is only a guide for maintaining the static GitHub + Cloudflare website.

Add a new Result

  1. Upload your file to assets/results/.
  2. Name it like result-013.png or result-014.mp4.
  3. Commit to GitHub.
  4. GitHub Actions will update assets/results/manifest.json automatically.
  5. Cloudflare Pages will deploy the updated site.

Add a new Review

  1. Upload your file to assets/reviews/.
  2. Name it like review-011.png or review-012.jpg.
  3. Commit to GitHub.
  4. The manifest will update automatically.

Video Posters

If you upload a video, you can add a poster image with the same number:

assets/results/result-014.mp4
assets/results/result-014-poster.jpg
