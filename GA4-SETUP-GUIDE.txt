GOLD HUNTER GA4 ANALYTICS PRO
Measurement ID: G-RMTE7H8TP1

DEPLOYMENT
1. Upload this complete folder/ZIP to GitHub or Cloudflare Pages.
2. Wait for Cloudflare deployment to finish.
3. Open goldhunter.site in an Incognito window and click a WhatsApp button.
4. In GA4 go to Reports > Realtime. Look for whatsapp_click.

KEY EVENT
Admin > Data display > Events
- Open Recent events.
- When whatsapp_click appears, click the star beside it to mark it as a Key event.
You may also create a key event named whatsapp_click before data arrives if your GA4 interface offers that option.

CUSTOM DIMENSIONS
Admin > Data display > Custom definitions > Create custom dimension.
Scope for every item below: Event.

SOURCE TRACKING
The script stores first-touch source in localStorage and records the current/last source.
Use campaign links such as:
https://goldhunter.site/?utm_source=instagram&utm_medium=social&utm_campaign=july_membership&utm_content=bio

Recommended utm_source values:
instagram, facebook, tiktok, xiaohongshu, google, whatsapp
