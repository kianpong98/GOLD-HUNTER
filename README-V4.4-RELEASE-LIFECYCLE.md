# Gold Hunter V4.4 — Release lifecycle

- Before release: Previous = latest published official Actual.
- Release day: Actual is shown only when official period exactly equals event releasePeriod.
- Release day: new Actual is never copied into Previous.
- Next Malaysia calendar day at 00:00: Actual + Forecast + Previous are archived into Last Release.
- After archive: current Forecast is cleared for the administrator to enter only the next forecast.
- Previous is fully automatic and read-only in Admin.
